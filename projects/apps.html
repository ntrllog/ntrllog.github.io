Notification Reminder

Download app (Notification.Reminder.apk) here

This is an app for creating reminders that can be easily seen. The lock screen is probably the most-viewed screen, so I wanted to put my reminders there. I was inspired to make this after I forgot to go to something I signed up for. (Oops!)

Copy Text

Download app (Copy.Text.apk) here

This is like a clipboard manager. It allows you to privately add and save strings of text so you can copy/paste for future use. I made this because I was tired of having to type out my password every time. #correcthorsebatterystaple

Bagels

Download app (bagels.apk) here

This is a numbers-deduction game. The game is also known as Pico Fermi Bagels. This was the first app I ever created on my own. It was inspired by Al Sweigart's Invent With Python.

Notepad

Download app (Notepad.apk) here

This is a text editor for writing and saving text files to your phone's external storage. I use this for making quick notes on the go or for writing when I don't have my laptop.

Time Difference

Download app (Time.Difference.apk) here

This is a calculator that calculates the number of hours and minutes between two times. I made this because I was going to high schools for observation hours and I had to log the number of hours and minutes. High schools have oddly specific schedules. Class is going to be 1.5 hours, but let's make it from 12:46 to 2:14, not 12:45 to 2:15.

Digits of Pi

Download app (Digits.of.Pi.apk) here

This is an app that displays the digits of pi (3 digits at a time). It could be used to memorize the digits of pi. I was inspired to make this by a high school math teacher at one of the high schools at which I volunteered. She knows 190 digits of pi (and can probably keep going).
